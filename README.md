# BMOSocketCpp
#Programa para demonstrar o uso de Socket TCP/IP Cliente-Servidor na linguagem C++. 

##Para testar basta: 
1) Executar o programa, 
2) Abrir uma aba no Terminal, 
3) Inserir o seguinte comando nesse terminal:        
 		telnet localhost 54000
4) Tudo o que voce digitar, será recebido pelo program que o printará no console, e enviado de volta pra exibir no terminal.
